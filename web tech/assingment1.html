Basic git commands

git init: Creates an empty Git repo in your current folder. It's like setting up a new project folder that Git can track.

git clone url: Downloads a copy of a repo from GitHub. Like downloading a project's entire history to your computer.

git add .: Tells Git to track all changed files. Think of it as taking photos of your current work.

git git commit -m "message": Saves your changes with a description. Like creating a checkpoint in a video game!

git push: Uploads your commits to GitHub. Similar to sharing photos to the cloud.

git pull: Downloads the latest changes from GitHub. Like updating your apps to get new features.
